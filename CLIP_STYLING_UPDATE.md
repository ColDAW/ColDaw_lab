# Clip 样式更新说明

## 🎨 更新内容

### 1. **渐变调色板实现**
为项目编辑页面的 clip 实现了基于轨道的渐变调色板系统：

```javascript
const TRACK_COLORS = [
  '#C9A511', // 金黄色
  '#53A21F', // 亮绿色 
  '#1CA231', // 绿色
  '#1CA26C', // 青绿色
  '#1CA2A2', // 青色
  '#297DA5', // 蓝绿色
  '#3C5BAD', // 蓝色
  '#553FAD', // 蓝紫色
  '#7932A9', // 紫色
  '#881BA3', // 深紫色
  '#A51AA3', // 紫红色
  '#A31C86', // 红紫色
  '#A31CA3', // 红紫色
  '#891BA3'  // 深紫红色
];
```

### 2. **轨道颜色分配**
- 每个轨道使用调色板中的一种固定颜色
- 同一轨道内的所有 clip 使用相同的颜色
- 轨道索引循环使用调色板：`TRACK_COLORS[trackIndex % TRACK_COLORS.length]`

### 3. **动态卡片样式**
参考 landing page 的动态卡片样式，为 clip 添加了：

#### 背景效果
- **渐变填充**: 从 90% 不透明度到 80% 不透明度的线性渐变
- **毛玻璃效果**: `backdrop-filter: blur(10px)`
- **移除状态**: 20% 不透明度的特殊渐变

#### 描边设计
- **基础描边**: 40% 不透明度的轨道颜色
- **微妙光晕**: 19% 不透明度的外发光效果
- **Hover 增强**: 31% 不透明度的强化光晕

#### 圆角和阴影
- **圆角**: 12px 圆角，更现代的设计
- **多层阴影**: 
  - 基础阴影：`0 1px 3px rgba(0, 0, 0, 0.12)`
  - 加深阴影：`0 1px 2px rgba(0, 0, 0, 0.24)`
  - 光晕效果：轨道颜色的半透明描边

### 4. **交互动画**
#### Hover 效果
- **3D 提升**: `translateY(-3px)` 向上移动
- **轻微缩放**: `scale(1.02)` 2% 放大
- **阴影增强**: 更深的阴影和更强的光晕
- **亮度提升**: `brightness(1.1)` 10% 亮度增加
- **层级提升**: `z-index: 10` 确保悬停元素在顶层

#### Active 效果
- **轻微压缩**: `translateY(-1px) scale(1.01)`
- **快速过渡**: 0.1s 的快速响应

### 5. **过渡动画**
使用 landing page 相同的缓动函数：
- **基础过渡**: `cubic-bezier(0.4, 0, 0.2, 1)` 0.2s
- **平滑响应**: 确保所有状态变化都有流畅的过渡

### 6. **文本样式优化**
#### ClipName 更新
- **字体大小**: 从 11px 增加到 12px
- **内边距**: 从 4px 8px 增加到 6px 10px  
- **颜色**: 95% 不透明度的白色，更清晰
- **阴影**: 增强的文本阴影效果
- **字间距**: -0.2px 的紧凑字间距
- **层级**: `z-index: 2` 确保文本在顶层

## 🎯 实现效果

1. **统一的视觉语言**: 与 landing page 的动态卡片保持一致的设计风格
2. **轨道差异化**: 每个轨道都有独特的颜色标识，便于区分
3. **专业的交互**: 悬停和点击都有流畅的反馈动画
4. **现代化外观**: 毛玻璃效果和多层阴影营造现代感

## 🔧 技术实现

- 替换了原来基于 `clip.color` 的 HSL 颜色系统
- 改为基于 `trackIndex` 的固定调色板系统
- 保留了所有现有的 diff 功能（添加、删除、修改状态）
- 保持了循环播放的虚线边框样式
- 兼容所有现有的交互功能
